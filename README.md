# LinkedIn Profile Status Analytics Dashboard

## Overview

Developed an interactive Power BI dashboard using a sample dataset created for learning and portfolio purposes. The dashboard analyzes professional profile trends based on OpenToWork status, Hiring activity, industry, location, experience level, and professional connections.

## Features

* KPI Cards for Total Profiles, OpenToWork Profiles, Hiring Profiles, and Average Experience
* Status Distribution Analysis
* Industry Analysis
* Location Analysis
* Experience Analysis
* Connections Analysis
* Interactive Slicers and Filters

## Tools Used

* Power BI
* DAX
* Microsoft Excel

## Dataset

This project uses a manually created sample dataset and does not contain real LinkedIn user data.
